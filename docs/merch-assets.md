# Merch Assets

The merch renderings live in `public/merch` so public pages can reference stable paths through Next.js.

These files are the current event-owner supplied renderings for Bubelpalooza merch. Keep the product artwork faithful to the supplied rendering. Background cleanup, transparency work, or format optimization is acceptable only when the product itself is not visually changed.
